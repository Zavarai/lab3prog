программа вычисляет интеграл функции f(x) = 2x³ + 2x² + 7 методом средних прямокгольников.
Основные функции:
calculate_integral — вычисляет интеграл методом средних прямоугольников.
F_antiderivative — первообразная для точного сравнения.
display_error_estimation — показывает точное значение и погрешность.
Интерфейс — текстовое меню с управлением стрелкамми вверх/вниз и Enter:
Ввод нижнего и верхнего пределов (a, b).
Ввод числа разбиений (n).
Вывод значения интеграла.
Вывод оценки погрешности.
Выход.
Проверки:
Убеждается, что a < b и n > 0.
